# ESAME_MCF_PALAZZETTI

## DIPENDENZE
Sarà necessario installare le dipendenze richieste: 

`pip install matplotlib`

`pip install numpy`

## ESECUZIONE
Si procede quindi a lanciare il programma nel terminale

`python3 simulazione_muoni.py`

Si procede quindi ad inserire i dati via via richiesti per procedere alla configurazione del modelo fisico del tracciatore di muoni

## NOTE TECNICHE
Risulta necessario, affinchè lo studio abbia un buon grado di attendibilità e considerevolezza:
- [ ] inserire un numero consistente di muoni
- [ ] inserire valori delle energie non troppo bassi.
